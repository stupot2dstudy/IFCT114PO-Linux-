# Bash Advanced Exercises - English Version

Run any script like this:

```bash
chmod +x exercise01.sh
./exercise01.sh
```

For exercise 1, test the default value:

```bash
./exercise01.sh
USER_INPUT="Andres" ./exercise01.sh
```

For exercise 5:

```bash
./exercise05.sh start
./exercise05.sh stop
./exercise05.sh something
```

For exercise 6, create the test file:

```bash
echo "test data" > datos.log
./exercise06.sh
```
